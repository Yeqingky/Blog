常见问题

同一页面中多次调用 图片一样

在同一网页页面中 多次调用 https://api.yppp.net/pe.php 图片一致

问题原因

浏览器缓存导致的问题 因为引用的URL完全一致 浏览器会复用缓存

解决办法

在URL中,添加任意不同的查询参数

示例

问题示例 如下 同样的URL 浏览器会直接复用缓存

调用
图片1
图片2

在URL后面添加任意不同的查询参数 让浏览器不要复用缓存

调用
图片1
图片2
